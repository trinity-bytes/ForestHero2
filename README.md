# ForestHero2

## Descripción

ForestHero2 es un juego desarrollado en C++. El objetivo principal del juego es controlar a un guardián del bosque para protegerlo y reforestarlo. El jugador deberá recolectar semillas y agua, plantar árboles y enfrentarse a enemigos para mantener el bosque saludable.

## Características (Inferidas del código)

*   **Jugabilidad:** Control del personaje Guardián, recolección de ítems (semillas, agua), plantación de árboles, sistema de puntuación, vidas, tiempo limitado.
*   **Elementos del juego:** Enemigos, aliados, power-ups (corazón), basura.
*   **Mecánicas:** Movimiento del personaje, disparo de semillas, interacción con elementos del entorno.
*   **Interfaz:** Menú principal, pantalla de juego, pantalla de créditos, pantalla de instrucciones, pantalla de derrota y victoria.
*   **Sonido:** Música de menú, música de juego, música para el tramo final, música de victoria y derrota.
